Adaptive Actuator Intelligence

AAI Research

C1 — Core Innovation

Learned Compensator

Lightweight neural network running inside the 20 kHz FOC ISR. Learns friction maps, cogging torque profiles, and nonlinear dynamics in real time to produce feed-forward compensation current that the PI controller never has to chase.

C2 — Online Learning

Online Micro-Learning

Continuous weight updates driven by prediction error between the NN output and measured torque disturbance. Uses a fixed-point Q15.16 inference path with rank-1 gradient updates that fit within the ISR timing budget.

C3 — Impedance Control

Impedance Shaping

Variable stiffness and damping profiles for contact-rich tasks. The AAI layer modulates virtual spring-damper parameters based on interaction force estimates, enabling compliant behavior without dedicated force/torque sensors.

C4 — Sim-to-Real

Sim-to-Real Bridge

NVIDIA IsaacSim domain randomization pipeline that trains policies across thousands of randomized motor parameters (friction, inertia, backlash, thermal), then deploys quantized weights directly to the STM32 CORDIC-accelerated inference engine.

Power Architecture

▼

Three-stage power conversion from 48V battery input:

StageICInputOutputType
1U2 LMR3650648V VBAT5V @ 600mABuck (560kHz)
2U4 TPS621725V3.3V @ 500mABuck (2MHz)
3U5 LP29923.3VA3.3V @ 250mALDO (analog)

The analog 3.3V rail (A3.3V) from U5 feeds the ADC reference and current sense amplifier references. It is isolated from digital 3.3V via ferrite beads FB1-FB4 to minimize switching noise coupling.
